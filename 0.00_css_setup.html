CSS Notes

Purpose of CSS

  • To give the outside of the site a unique look and feel
  • Alter colors,fonts, backgrounds, ect
  • To implement some effects and light animations

CSS statments

Four main components include the following:

  1. Declaration:the statement as a whole with curly braces
  2. Selector:the name of the part you want to style
  3. Property:the properties to edit(e.g.width,heigth,background,ect)
  4. Value:numbers, colors, urls- values we assign
